## 🎨 Визуальные и звуковые особенности (глобально)

| Элемент | Реализация |
|---------|-----------|
| **Цветовая схема** | Тёплые тона: золото (#D4AF37), бордо (#800020), кремовый (#FFFDD0), тёмно-синий (#0F1B3A) для акцентов |
| **Шрифты** | Заголовки: `Cormorant Garamond` (церковнославянский стиль), Основной: `PT Sans` / `Roboto` для читаемости |
| **Иконки** | Линейные, с элементами православной символики (крест, свеча, купол) |
| **Анимации** | Плавное появление, мягкое мерцание свечи, эффект "благовеста" при добавлении в корзину |
| **Звуки** | • Фоновое пение (опционально, выкл. по умолчанию) <br> • Уведомления: тихий звон колокольчика <br> • Успешный заказ: короткий благовест |
| **Доступность** | Поддержка скринридеров, контрастный режим, увеличение шрифта |

## 🗂️ Структура маршрутов (Frontend)

| Маршрут | Страница | Доступ | Примечание |
|---------|----------|--------|-----------|
| `/` | Главная | Публичный | С цитатой дня и виджетами |
| `/catalog` | Каталог | Публичный | С фильтрами, включая "постный режим" |
| `/catalog/:category` | Категория | Публичный | Иконы, книги, свечи и т.д. |
| `/product/:id` | Товар | Публичный | С церковными рекомендациями |
| `/cart` | Корзина | Публичный | + случайная цитата из Псалтыри |
| `/checkout` | Оформление | Авторизованный | С опцией освящения заказа |
| `/profile` | Личный кабинет | Авторизованный | Заказы, свечи, настройки |
| `/candle` | Виртуальная свеча | Публичный | Отдельная духовная функция |
| `/baptism-name` | Генератор имён | Публичный | По святцам на дату |
| `/live` | Трансляция | Публичный | Если идёт служба |
| `/quotes` | Цитаты / Псалтырь | Публичный | Справочный раздел |
| `/about` | О лавке | Публичный | Миссия, реквизиты, контакты |

---

> 💡 **Принципы проектирования**:  
> 1. **Уважение к традиции** — никакой излишней "игривости", сдержанный, благоговейный дизайн.  
> 2. **Доступность** — сайт должен быть удобен и пожилым прихожанам, и молодёжи.  
> 3. **Духовная польза** — даже коммерческие элементы (корзина, карточка товара) несут просветительскую функцию.  
> 4. **Тактичность** — звуковые эффекты отключаемы, "постный режим" — рекомендация, а не принуждение.

Этот макет можно использовать как основу для передачи дизайнеру и разработчикам. При необходимости — могу детализировать любой экран или добавить прототипы состояний (загрузка, ошибка, пустой список).
