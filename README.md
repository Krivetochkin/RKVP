# Святое Торжище
Ссылка: https://svyattorg.by/

**Святое Торжище** — это специализированный интернет-магазин, предлагающий широкий ассортимент православной продукции для верующих и интересующихся культурой. Платформа предназначена для удобного поиска и приобретения икон, духовной литературы, церковной утвари и подарков с религиозным смыслом. Мы стремимся сделать православные товары доступными для покупателей по всей стране, обеспечивая качественный сервис и достоверность информации о каждом изделии. Сайт также служит просветительским ресурсом, помогая пользователям правильно ориентироваться в выборе церковных принадлежностей.

## Целевая аудитория

*   Практикующие православные христианы, желающие приобрести товары для личной молитвы и церковного быта.
*   Люди, подбирающие тематические подарки к религиозным праздникам (Пасха, Рождество, Крещение, именины).
*   Интересующиеся православной культурой, историей церкви и духовной литературой, включая студентов и исследователей.

## Ключевой функционал

*   **Каталог товаров:** Удобная навигация по категориям (иконы, книги, свечи, одежда, утварь) с фильтрами по цене, размеру, материалу и назначению.
*   **Корзина и оформление заказа:** Безопасная оплата онлайн, выбор способа доставки и отслеживание статуса заказа в личном кабинете.
*   **Личный профиль пользователя:** История покупок, сохранение избранных товаров в «Отложенное» и управление адресами доставки.
*   **Информационные карточки товаров:** Подробное описание с учетом церковных традиций, рекомендации по выбору иконописных стилей и указание размеров для церковной одежды.
*   **Кнопка «Поставить свечу онлайн»:** Пользователь сможет зажечь виртуальную свечу.
*   **Генератор имен для крещения:** Если родители не могут определиться, то генератор поможет выбрать имя святого из календаря на текущую дату.
*   **Уникальное звуковое и визуальное сопровождение:** В качестве звукового сопровождения будет возможность прослушать православные музыкальные произведени, пока пользователь выбирает товар. В качестве звука уведомления будет звон колоколов.
*   **Случайная цитата в корзине:** Пока пользователь оформляет заказ, внизу отображается рандомная мудрая мысль или стих из Псалтыри. Поднимает настроение и настраивает на нужный лад.
*   **Прямая трансляция службы:** Прямо на сайте можно будет просматривать прямую трансляцию службы, если таковая проходит сейчас.
*   **Фильтр товаров по узким параметрам:** Фильтр скрывает на сайте товары, содержащие кожу (например, дорогие переплеты из натуральной кожи), если человек выбрал категорию "пост".
